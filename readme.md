# IOM Campus

A prototype learning management system for IOM students.
[Watch a demo video](https://youtu.be/lz5zbfwN4yE?si=-xthPFuLTZC5ycOz)


## Project Purpose

IOM Campus is a Django-based platform designed to facilitate student learning and course management for IOM students.

## Technology Stack

- **Backend**: Django
- **Language**: Python

## Key Features

- Student management
- Course management
- Lecture video sharing
- Study materials and sheet sharing
